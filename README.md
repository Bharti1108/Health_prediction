# Health_prediction
 A basic health prediction application using Python as the backend and streamlit as frontend technology .
 Health Prediction Application using patient blood test results and an external AI/ML Health
 API.
 The application should collect and manage the following information for each contact:
• Full Name
• Date of Birth
• Email Address
• Glucose
• Haemoglobin
• Cholesterol
• Remarks(Generated from AI)
